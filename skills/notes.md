# Quick Notes

Capture quick notes, thoughts, and ideas.

## Triggers
- note
- remember
- jot down
- save this
- thought

## Data Schema
```json
{
  "type": "text",
  "fields": ["content", "tags"],
  "searchable": true
}
```

## Actions
- add: Save a new note
- list: Show recent notes
- search: Find notes by keyword
- tags: List all tags

## Examples
- "note: call dentist tomorrow"
- "remember to buy milk"
- "jot down: project idea - meal planner app"
- "show my notes"
- "search notes for dentist"
