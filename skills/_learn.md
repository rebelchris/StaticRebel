# Learn New Skill

Meta-skill for teaching the companion new capabilities.

## Triggers
- learn
- teach
- new skill
- track new
- create tracker

## Description
This is a meta-skill that allows users to teach the companion new things to track or do. When invoked, the companion should ask clarifying questions to understand:
1. What to call the skill
2. What kind of data to track (numbers, text, activities)
3. What triggers should activate it
4. What actions it should support

## Flow
1. User says "teach you to track X" or "learn X"
2. Companion asks: "What should I call this skill?"
3. Companion asks: "What kind of data should I store? (numbers, text, activities)"
4. Companion asks: "What words should trigger this skill?"
5. Companion creates the skill file and confirms

## Examples
- "teach you to track my coffee intake"
- "learn to track my reading"
- "new skill: meditation timer"
- "can you track my screen time?"
