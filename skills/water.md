# Water

Track daily water intake to stay hydrated.

## Triggers
- water
- drank
- hydrate
- glass

## Data Schema
```json
{
  "type": "numeric",
  "unit": "ml",
  "defaultAmount": 250
}
```

## Actions
- log: Record water intake
- today: Show today's total
- history: Show recent entries
- goal: Check progress
